Show one or more "Custom Grouped Run's Chart (Heatmap)" instances on the job's vManager Charts page.

Each chart groups the runs of the build's resolved vManager session(s) by a chosen RUN_LEVEL attribute and renders the result as a heat-map: one column per build (most recent on the left), one row per distinct attribute value (most recent on top), each cell coloured by the count of entities in that group.

Add one chart per attribute you want to slice by — e.g. one chart for First Failure Description, another for Test Group, another for Computer, etc.
